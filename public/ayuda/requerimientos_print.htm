1.2. Requerimientos

Requerimientos de Software
1. Sistema Operativo:
  • Windows: Windows 10 o superior.
  • macOS: macOS Sierra o superior.
  • Linux: Distribuciones modernas como Ubuntu 20.04 o superior.
  • Dispositivos móviles: iOS 12 o superior, Android 9.0 o superior.
    2. Navegador Web: El sistema web está optimizado para los siguientes navegadores:
  • Google Chrome (última versión).
  • Mozilla Firefox (última versión).
  • Microsoft Edge (última versión).
  • Brave (última versión).
    Se recomienda tener la última versión del navegador para garantizar compatibilidad total con el sistema.
    3. Software de soporte:
  • Adobe Acrobat Reader o cualquier otro lector de PDF (si el manual está en formato PDF).
  • Editor de texto (si el manual está en formato HTML, para edición o personalización).
    4. Conexión a Internet:
  • Una conexión a internet estable para acceder al sistema web y al manual en línea. El sistema está basado en la web, por lo que requiere acceso a internet para funcionar correctamente.
Requerimientos de Hardware
1. Requerimientos Mínimos:
  • Procesador: CPU de 1.8 GHz o superior (Intel Core i3, AMD Ryzen 3, o equivalente).
  • Memoria RAM: 4 GB o más.
  • Espacio en Disco Duro: 10 GB de espacio libre.
  • Tarjeta Gráfica: Compatible con resolución mínima de 1280x720 píxeles.
  • Conexión a Internet: Conexión mínima de 1 Mbps para acceso a la web.
    2. Requerimientos Recomendados:
  • Procesador: CPU de 2.5 GHz o superior (Intel Core i5, AMD Ryzen 5, o equivalente).
  • Memoria RAM: 8 GB o más.
  • Espacio en Disco Duro: 20 GB de espacio libre o más.
  • Tarjeta Gráfica: Compatible con resolución de 1920x1080 píxeles o superior.
  • Conexión a Internet: Conexión de 10 Mbps o superior para un rendimiento fluido del sistema.
    3. Dispositivos Móviles:
  • Teléfonos móviles: Dispositivos con Android 9.0 o superior o iPhone con iOS 12 o superior, con al menos 2 GB de RAM.
  • Pantalla: Resolución mínima recomendada de 720p para asegurar la legibilidad del sistema.
