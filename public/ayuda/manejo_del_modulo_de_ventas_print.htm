2.2. Manejo del módulo de ventas

En la pantalla de control de ventas de productos, tenemos las opciones que se pueden observar en la siguiente imagen.
 

Botón para agregar nueva venta

Al hacer click en este botón, se dirigirá a la pantalla donde se podran registrar todos los productos de la venta.

Botón para ver factura

Al hacer click en este botón, se dirigirá a la pantalla donde podra visualizar el detalle de la venta en la presentación de una boleta electrónica, dicha boleta también se podrá imprimir.

Botón para cancelar venta

Al hacer click en este botón la venta de cancelará, por lo que se revertirán los cambios realizados en la venta registrada.

Casilla para buscar una venta

En esta casilla puede digitar la fecha, el nombre del cliente o el código para poder buscar la venta que desea.
