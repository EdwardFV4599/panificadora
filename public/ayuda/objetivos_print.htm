1.1. Objetivos

  • Facilitar la interacción con el sistema: Proporcionar a los usuarios una guía clara y accesible para utilizar el sistema web de la panificadora "Coronel" de manera eficiente y efectiva.
  • Capacitación de los usuarios: Instruir a los empleados de la panificadora en el uso de todas las funcionalidades del sistema, permitiéndoles realizar sus tareas diarias sin dificultades.
  • Optimizar la gestión operativa: Asegurar que los usuarios comprendan cómo usar el sistema para gestionar inventarios, ventas, pedidos, producción y control de clientes.
  • Mejorar la experiencia del usuario: Ofrecer un documento intuitivo que respalde la experiencia del usuario, reduciendo la necesidad de asistencia técnica constante.
  • Garantizar la seguridad en el uso: Enseñar a los usuarios cómo proteger la información sensible dentro del sistema y cómo usar las funcionalidades de manera segura.
